# Problem 1
Instructions:
Write separate C programs for each of the problems in the document.
Number the files according to the numbers of the problems, like 1.c, 2.c, etc.
Then zip the c files (using WinRAR or 7-zip) and upload.
Delete the other files with extensions like.o,.obj, or.exe. Do not include them in the assignment.
Name the zip file with your first name and ID number, for example, prottoy_251-134-007.
